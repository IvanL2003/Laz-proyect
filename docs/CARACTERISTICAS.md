# Características del Lenguaje Laz

Laz es un lenguaje de programación experimental diseñado para ser simple, expresivo y fácil de extender. A continuación se describen sus principales características:

## 1. Sintaxis Simple
- Inspirada en lenguajes modernos y scripting.
- Soporte para variables, funciones, condicionales y bucles.

## 2. Tipado Dinámico
- Las variables no requieren declaración de tipo.
- El tipo se determina en tiempo de ejecución.

## 3. Funciones de Primera Clase
- Las funciones pueden ser asignadas a variables, pasadas como argumentos y retornadas.

## 4. Estructuras de Control
- Soporte para `if`, `else`, `while`, y `for`.

## 5. Operaciones con Archivos CSV
- Lectura y escritura de archivos CSV de forma nativa.

## 6. Interoperabilidad con C y LLVM
- Generación de código C y LLVM para ejecución eficiente.

## 7. Formateador de Código
- Herramienta integrada para formatear el código fuente.

## 8. Sistema de Errores
- Manejo de errores con mensajes claros y localización precisa.

## 9. CLI Intuitiva
- Interfaz de línea de comandos para ejecutar, compilar y formatear código.

## 10. Extensible
- Arquitectura modular que permite agregar nuevas funcionalidades fácilmente.

## 11. Ejemplos Incluidos
- Ejemplos de uso en la carpeta `examples/`.

## 12. Documentación
- Documentación detallada en la carpeta `docs/`.
